\doxysubsection{Detailed Description}
The structure that defines an extended virtual joystick description

The caller must zero the structure and then initialize the version with {\ttfamily SDL\+\_\+\+VIRTUAL\+\_\+\+JOYSTICK\+\_\+\+DESC\+\_\+\+VERSION} before passing it to \mbox{\hyperlink{_s_d_l__joystick_8h_a4166ce7dc20f4fa5dfd8f09e1d1ed04d}{SDL\+\_\+\+Joystick\+Attach\+Virtual\+Ex()}} All other elements of this structure are optional and can be left 0.
