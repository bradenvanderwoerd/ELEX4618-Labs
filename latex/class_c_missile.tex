\doxysubsection{Detailed Description}
Missile class for asteroid game. 

This class represents a missile in the game that is fired by the player\textquotesingle{}s ship. It inherits from \mbox{\hyperlink{class_c_game_object}{CGame\+Object}} and contains methods for moving the missile.
